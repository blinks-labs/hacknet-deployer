# Optimum Hacknet Deploy

Clone repo `git clone git@github.com:getoptimum/hacknet-deploy.git`

Check `README.md` under `docs` folder for a detailed explanation of the experiments to run. 

You will need to be able to run Makefiles. This `Makefile` contains information about how stop and start docker containers that will be required duirng the hackathon, namely the `p2pnode` and the `proxy`. 

You will also need a `docker-compose.yml` file to start those services.
